## Open Ai Gym experiment

- Anaconda 3
- OpenAI Gym
- Tensorflow 2.5

Per eseguire il notebook è necessario avviare Jupyter tramite Anaconda, selezionare il notebook OpenAiExperiment.ipynb ed eseguire le celle di codice presenti al suo interno.
